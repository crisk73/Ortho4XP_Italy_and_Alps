Pre-built Ortho4XP v1.40 basepack tiles for Italy and Alps to be used with Autoortho or X-Plane Map Enhancement.

Usage:

For Autoortho: copy and overwrite the folder **z_ao_eur** into [XP12 path]\Custom Scenery\z_autoortho\scenery\

For XPME: copy and overwrite the **subfolders** contained in z_ao_eur into \[XPME basepackage folder path]\XPME_Europe\ and rename the folder **"texture"** into **"_texture"**.
